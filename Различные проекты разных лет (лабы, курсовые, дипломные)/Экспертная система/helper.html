Если вы эксперт:

Используемый операторОписание
<n< (between)

Используется для поиска значения в данном диапазоне. Диапазон указывается как пара значений 0;24 или 1990;2004. Пример в базе знаний 

<condition left="Информация о предпочтениях.Возраст аудитории" rel="between" right="17;120"/>

< (less)

Используется для поиска значения меньшего, чем указанное. Пример в базе знаний 

<condition left="Информация о предпочтениях.Год выпуска" rel="less" right="1990"/>

> (greater)

Используется для поиска значения большего, чем указанное. Пример в базе знаний 

<condition left="Информация о предпочтениях.Год выпуска" rel="greater" right="1990"/>

=(equals)

Используется для поиска значения точно совпашего с указанным значением. Пример в базе знаний 

<condition left="Информация о предпочтениях.Страна выпуска" rel="equals" right="США"/>
